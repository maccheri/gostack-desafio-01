# Gostack Desafio 1: Conceitos do NodeJS
Desafio de NodeJs - GoStack - Rocketseat

## Instalar o projeto

`
  yarn install
`

## Rodar o projeto

`
  yarn dev
`

## Libs
O projeto utiliza somente 2 libs:
  * Express - para construir as chamadas REST.
  * Nodemon - hotreload no backend.
